O que são Clients?

Definição:

É o lado que acessa atraves do protocolo http o conteudo hospedado em um servidor.
Exemplo:Smartphone, PC, Tablets.

Com uma conexão estabelecida a um server(já vamos falar deles) é possivel então receber arquivos estáticos - HTML, CSS, JS e/ou imagens/áudios. Nessa ponta da conversa temos também os famosos Browsers, que armazenam esses arquivos solicitados em cache para acelerar uma posterior requisição.

Browsers

Definição: Programas criados por empresas para abrir/executar arquivos específicos.
EX: Chrome, Firefox e Safari.

Principais Características:

  • Segue padrões W3C (porém possuem diferenças de interpretação de arquivos)
  • Gratuitos

Aplicação Web

Definição: Soluçao/produto que utiliza a internet como meio de comunicação entre Cliente x Server e sem precisar realizar a instalação da aplicação.
EX: YouTube, Linkedin ou Facebook (desde que usados pelo navegador).

E lembre-se:

"Trabalhar com a internet não é o futuro, Já estamos vivendo nele!"

O que são Servers?

Definição:

Lado em que se fornece serviços/conteúdo da internet ou em uma LAN(Local Area Network). Ao restringir acesso a rede externa a um servidor ele atua em uma INTRANET.

Geralmente são máquinas mais robustas, o conjunto de servidores é denominado de Data Center.

Principais serviços oferecidos são:

  1. Arquivos
  2. Segurança (proxy/firewall)
  3. Streaming
  4. E-mail
  5. Web

Proxy

Definição: serviço que fornece regras de acesso a internet. Função de restringir a navegação a determinados sites em uma rede local. Regras são estabelecidas pela empresa/organização que implementou.

Firewall

Definição: Servidor dedicado a segurança que faz o controle de acesso e informações recebidos externamente em uma rede local.

Servidor Web

Definição: Responsável por tratar requisições HTTP ( feitas pelo cliente ) e encaminhar respostas/arquivos correspondentes.

Hospedagem de sites

Ao se criar um site é necessário hospedá-lo em um local para que usuários possam acessar atráves da internet, ou seja precisamos de um Servidor Web. Para isso podemos contratar planos onde empresas oferecem esse serviço.

Para disponibilizar um site na grandiosa internet precisamos ter um domínio registrado, ou em termos mais simples, um nome para nosso site. Aqui no Brasil podemos fazer esse registro através do site Registro.br.

Uma vez que o domínio já está cadastrado podemos então assinar um plano de hospedagem. O próximo passo seria então incluir o servidor DNS, disponibilizado pela empresa contratada. Não se assuste com a sigla, esse tal DNS nada mais é que um tradutor de domínios ( nome do site) em IPs(endereço lógico do site na internet).Geralmente empresas que oferecem hospedagem costumam disponibilizar também um servidor de e-mail para uso profissional

Por último e não menos importante, como podemos fazer para incluir os arquivos do nosso site nesse servidor web ? muito simples! Utilizando o protocolo FTP( isso mesmo, mais uma sigla). Para fazermos esse envio a empresa nos fornece as credencias para usar esse serviço. Ai basta acessarmos o endereço fornecido e logar com o usuário e senha informados, feito isso já podemos enviarmos nossos arquivos HTML, CSS, JS que compõe nosso site.

Como é uma estrutura básica de um HTML?

Definição:

HTML é uma linguagem de marcação utilizada para estruturar e organizar semanticamente elementos em um página web.

O principal elemento de marcação e a Tag, sendo sempre demilitado pelos simbolos < > para abertura e para fechamento, sendo esse último opcionais em alguns elementos.

Geralmente o arquivo HTML principal de um página é nomeado como index.html

Estrutura básica de um arquivo HTML

  1. Tag html
    • Engloba todos os elementos
  2. Tag head
    • carregamento de metadados ( não visiveis ao usuário geralmente).
      EX:Titulo da pág.links de estilo e scripts, idioma.
  3. tag body
    • Elemento visiveis do site.
      EX:Texto, imagens, links

OBS: Uma tag pode possuir atributos, podendo ser mais especificos ou mais genéricos.
EX: atributo id e atributo type da tag input.
